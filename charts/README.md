# charts — Satellite Finance / Data

Ce dossier est un satellite orienté data/finance : graphiques, indicateurs, vues d’analyse, et composants liés à l’observation de marché.

## Rôle

- Visualisation de données (charts)
- Surfaces de lecture/monitoring
- Module autonome, consommable depuis le Hub

## Stack

- Vite + React
- TypeScript
- lightweight-charts / Three.js (selon les vues)

## Développement

```bash
npm install
npm run dev
```

Lint / build :

```bash
npm run lint
npm run build
```
