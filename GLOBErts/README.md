# GLOBErts — Satellite Monde / Carte

Ce dossier est un satellite orienté “monde” : visualisation type globe/carte, couches d’information, lecture de contexte géographique.

## Rôle

- Affichage globe/carte (Three.js + librairies globe/carto)
- Panneaux d’UI et télémétrie
- Module autonome, consommable depuis le Hub

## Stack

- Vite + React
- TypeScript
- Three.js
- maplibre-gl / react-globe.gl

## Développement

```bash
npm install
npm run dev
```

Lint / build :

```bash
npm run lint
npm run build
```
