# gnosis-app — Satellite Savoir

Ce dossier est un satellite orienté “connaissance” : contenus, cartes de savoir, interfaces de consultation. Il est conçu pour être accessible depuis le Hub 3D comme une extension de l’univers (lore, wiki, guides).

## Rôle

- Surface de contenu (savoir, références, pédagogie)
- Expérience UI propre, indépendante, mais alignée avec l’écosystème

## Stack

- Next.js (App Router) + React
- TypeScript
- Tailwind

## Développement

```bash
npm install
npm run dev
```

Lint / build :

```bash
npm run lint
npm run build
```
